Proyecto de scraping web orientado a la extracción automatizada de datos desde portales inmobiliarios.
El sistema obtiene información estructurada a partir de contenido HTML dinámico, procesa los datos y los normaliza para su almacenamiento o consumo posterior.

El desarrollo se enfoca en comprender la estructura real de los sitios web, manejar variaciones en el DOM y asegurar la consistencia de los datos extraídos.
